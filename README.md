# なぜ作ったのか？
学生が主催するイベントにて、Webページからの応募をさせたいという思いがあり、LPページが欲しかったから。

3日

# 作った感想
イベント開催までの期間がとても短かったため、どのような見た目で、どのような内容にするかを学生団体と一緒になって製作を最短で終わらせた。
希望通りのイベント参加人数を確保できたのでとてもよかった。
